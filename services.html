    S3(Simple Storage Service)

    S3:Amazon S3 is an object storage service, which means it allows you to store and retrieve data as objects. Objects can include data files, images, videos, documents, and more. Each object is stored in a container called a "bucket."

    EC2(Eleastic Cloud Computing)

    Amazon Elastic Compute Cloud (EC2) is a web service that provides resizable compute capacity in the cloud, allowing users to run virtual servers (instances) for various computing workloads.

    EC2 allows businesses to run applications on the public cloud. An EC2 instance is a virtual server. Developers can use EC2 to build applications with great scalability.

    Creating of EC2 Process

    To create an application using resources stored on S3, you'll need to create an EC2 instance. You can use an Amazon Machine Image (AMI) to create a new instance. An AMI is a template that is used to create a new instance.

    VPC(Virtual Private Network)

    Amazon Virtual Private Cloud (VPC) is a virtual network in the AWS cloud that you can use to launch AWS resources. It's a foundational AWS service, which means that other AWS services, like Elastic Compute Cloud (EC2), can't be accessed without a VPC network.

    You can specify an IP address range for the VPC, add subnets, add gateways, and associate security groups. You can access resources and applications through IPv4 or IPv6.

    Lambda

    AWS Lambda is a serverless computing service from Amazon Web Services (AWS). It allows users to run code for applications or backend services without provisioning or managing servers.

    AWS Lambda executes code in response to events in AWS services, such as adding or removing files in an S3 bucket, updating Amazon DynamoDB tables, or HTTP requests from Amazon API Gateway.

    SNS(Simple Notification Service)

    Amazon Web Services Simple Notification Service (AWS SNS) is a cloud service that sends push messages from applications to subscribers. SNS is a fully managed messaging service that supports both application-to-application (A2A) and application-to-person (A2P) communication. It can send messages through email, SMS, and application notifications.

    SNS is a push-based messaging service, which means that a publisher sends a message to a subscriber immediately instead of storing the message. SNS stores all messages across multiple availability zones to prevent loss.

    SQS(Simple queue Service)

    SQS stands for Simple Queue Service. It is a web service from Amazon that provides access to a message queue. SQS is a high-performance service that offers queues to support message transactions between applications or among the components of an application

    Amazon SQS is a web service that gives you access to a message queue that can be used to store messages while waiting for a computer to process them.

    Kinesis

    Kinesis is a platform on AWS that sends your streaming data. It makes it easy to analyze load streaming data and also provides the ability for you to build custom applications based on your business needs.

    RDS(Relational Database Services)

    Amazon Relational Database Service (RDS) is a web service that helps you set up, operate, and scale relational databases in the AWS Cloud.

    RDS supports six database engines:

    Amazon Aurora: It is a MySQL-Compatible relational database with five times performance.

    Amazon DynamoDB: It is a fast and flexible NoSQL database service.

    Amazon Redshift: It is a petabyte-scale data warehouse service.

    Amazon Elasticache: It is an in-memory cache service with support for Memcached and Redis.

    AWS Database Migration Service: It is a service that provides easy and inexpensive to migrate your databases to AWS cloud.

    API(Application Programer interface)

    Amazon API Gateway is a service from AWS that helps developers create, publish, maintain, monitor, and secure APIs.

    API Gateway can be used to create REST, HTTP, and WebSocket APIs. API Gateway can be used to create APIs that access AWS or other web services, as well as data stored in the AWS Cloud.
