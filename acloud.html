About cloud

Read this:

cloud is a network which is located at a remote location we access it through the internet.

It is used to store data, retrieve data, backup data. We store files, videos, audios, run applications with low cost.

Types of Cloud

There are total four types of clouds are there:

PUBLIC, PRIVATE , HYBRID, COMMUNITY

PUBLIC: Public cloud is open to all to store and access information via the Internet using the pay-per-usage method

EXAMPLES:AWS,GLOOGLE CLOUD PLATEFORM,MICROSOFT AZURE.

PRIVATE: Private cloud is also known as an internal cloud or corporate cloud. It is used by organizations to build and manage their own data centers internally or by the third party.

EXAMPLES:AWS,GLOOGLE CLOUD PLATEFORM,MICROSOFT AZURE, Oracle Cloud,IBM.

HIBRID: Hybrid Cloud is a combination of the public cloud and the private cloud. we can say:

Hybrid Cloud = Public Cloud + Private Cloud

COMMUNITY: a group of several organizations to share the information between the organization and a specific community. THis are used by Government sectors,and some organisations
